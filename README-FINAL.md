## How It Works

### Conversation Mode
1. Select a topic (Sport, Viaggi, Identità, Opinioni)
2. Choose difficulty (Facile/Medio/Difficile)
3. Choose answer type (Breve/Elaborata/Entrambe)
4. See question and timer
5. Speak your answer out loud
6. Check hint or see example answers
7. Next question

### Grammar Drills Mode
1. Select a category (Verbi, Articoli, etc.)
2. Choose a specific drill
3. See fill-in-the-blank exercise
4. Type your answer
5. Check answer for instant feedback
6. See grammar rules if needed
7. Next exercise

## Usage Tips

### For Conversation Practice:
1. Read the question out loud
2. Think in Italian (don't translate)
3. Use the timer to simulate real conversation
4. Try short answers first for quick practice
5. Try long answers to build fluency
6. Check hints if stuck on structure
7. Review examples to learn new phrases

### For Grammar Drills:
1. Read the rules first
2. Type your answer
3. Don't worry about mistakes
4. Review correct answers even if right
5. Repeat drills until automatic
